# 2026년 1월 21일 회의록

## 📋 회의 개요
AI 기반 의류 코디 추천 서비스 프로젝트의 기술적 구현 방식, 데이터 처리, 그리고 멘토 피드백을 중심으로 진행된 회의 내용을 정리했습니다.

---

## 🌅 오전 강사님 피드백

### 옷장 등록 방식 간소화
- **문제점**: 첫 허들이 너무 높음
- **해결 방안**:
  - 책장을 찍어서 한 번에 처리하는 것처럼, 옷장을 찍어서 한 번에 옷을 넣어주는 작업
  - 정말 매일 입는 옷을 넣어주는 등 간단하게 접근
  - 옷을 많이 등록하면, 더 좋은 퀄리티의 답변을 제공받을 수 있음
- **추가 고려사항**: 비용 체크 필요

## 🌆 오후 추가 회의 및 IUI 피드백

### 1. 옷에서 추출하는 특징 (Feature Extraction)

- **색상** (Color)
- **상하의 여부** (Top/Bottom)
- **두께** (Thickness)
- **옷 소재** (Material/Fabric)
- **길이** → 반팔/긴팔
- **계절감** → 여름/겨울
- **핏** → 오버핏, 스트레이트 등
- **형태** → 스웨터, 후드티, 맨투맨, 파카
- **질감, 패턴, 무드** (Texture, Pattern, Mood)

### 2. AI Agent가 활용하는 맥락 (Context)

#### 2.1 날씨 정보
- 기온, 계절 등 날씨 데이터 활용

#### 2.2 대화 이력 (Conversation History)
- **Short-term memory**: 최근 대화 내용
- **Long-term memory**: 장기적인 대화 패턴
- **목적 추출**: 면접, 결혼식, 스키장 등 → 대화 이력에서 장소와 시간을 자동 추출

#### 2.3 옷장 데이터
- 옷에서 뽑아낸 특징들을 데이터베이스화

#### 2.4 사용자 기분 (Mood)
- 사용자의 현재 기분 상태

#### 2.5 코디 히스토리 (Calendar/History)
- **설계 변경**: 개인 일정 등록은 제외
  - 직접 등록하는 것이 큰 피로감
  - 대신 코디 히스토리만 기록
- **용도**: 겹치지 않은 코디를 하기 위한 데이터
- **기록 내용**: 일정이 아닌 실제 코디 기록

#### 2.6 코디 목적 (Purpose)
- 사용자가 코디를 요청하는 목적

### 3. 추가 개선 사항

- **초기 사용자 가이드**: 첫 채팅에 "이렇게 써라"는 예시를 보여줌
- **RAG 최적화**: 질문에 따라 관련 없는 정보를 필터링하는 기능
- **Agent 추론 방식**: 육하원칙(Who, What, When, Where, Why, How)을 기준으로 사고하도록 설계
- **재질문 정책**: 최대 2회, 사용자가 답변을 요청할 수도 있음

## 💻 기술 구현 논의

### 이미지 처리 방식

#### 초기 아이디어
- 행거(Hanger)에 걸린 옷 사진을 하나씩 찍어 객체를 분류(Classification)
- 배경 제거(누끼 따기) 처리

#### 논의 사항
- **쟁점**: 단순 분류 모델(Classification) vs LLM(거대언어모델) 활용
- **결론**: 
  - 옷의 특징(색상, 패턴, 소재, 핏, 계절감 등)을 추출하는 데 있어
  - 단순 분류보다는 **LLM(GPT-4o 등)을 활용**해 이미지를 분석하고 텍스트 태깅을 하는 것이 더 효율적

### 서비스 핵심 기능

1. **내 옷장 기반 추천**
   - 사용자가 가진 옷을 기반으로 코디 추천

2. **맥락 기반 추천**
   - 날씨(기온, 계절)
   - TPO(시간, 장소, 상황)
   - 사용자의 최근 스타일

3. **2단계 필터링 방식**
   - 1차: Rule-based 필터링 (날씨나 계절에 맞지 않는 옷 제외)
   - 2차: LLM 기반 코디 제안

## 🎯 MVP(최소 기능 제품) 범위

### 타겟
- **우선순위**: B2C(일반 사용자)
- **확장성**: 비즈니스 확장성 고려

### 핵심 기능 플로우
```
사용자가 옷장 사진 등록 
  → AI가 옷의 특징을 분석하여 데이터베이스화 
  → 외출 전 상황(날씨, 일정 등)에 맞는 최적의 코디 추천
```

팀은 현재 **"사용자의 옷장 사진을 분석해 TPO와 날씨에 맞는 코디를 추천해주는 AI 서비스"**를 기획 중이며, 기술적으로는 **LLM을 활용해 옷의 특징을 추출하고 추천하는 방식**을 채택했습니다.
